# Project Title
    My portfolio

## Description
    This is a simple static portfolio created using HTML and CSS.
    It includes sections like About, Skills, Projects, and Contact.
    The aim is to showcase basic web development skills.
    
## Features
- Navigation bar with internal links
- About section with profile picture
- Skills table
- Project showcase section
- Contact form
- Back-to-top button
-Responsive sections with smooth layout

## Technologies Used
- HTML
- CSS
- VS Code
- Git and GitHub
- Google Fonts (Poppins)

## How to Run the Project
1. Download or clone the repository.
2. Open the project folder.
3. Open **index.html** in any web browser.
4. No additional installation required.
